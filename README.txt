No nosso projeto utilizámos uma API externa do site: https://api-sports.io/documentation/basketball/v1
Esta API fornece informações sobre as estatísticas e resultados de vários desportos. 
Neste projeto usamos os widgets que são disponibilizados pela API relativamente aos resultados/jogos do dia e à tabela de classificações da NBA.
Tendo apenas 100 chamadas diárias à API, para evitar fazer uma chamada cada vez que o index é aberto, esses widgets estão comentados. 
Para ativá-los, é necessário remover os comentários do código.
Alteramos o CSS das imagens de modo a não ficarem desformatadas.
