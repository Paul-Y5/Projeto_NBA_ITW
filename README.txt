No nosso projeto utilizámos uma API externa do site: https://api-sports.io/documentation/basketball/v1
Esta API fornece informações sobre as estatísticas e resultados de vários desportos. Neste projeto usamos os resultados/jogos do dia e a tabela in live da NBA.
A forma de aplicação desta API no nosso projeto foi realizada de forma similar à já utilizada API fornecida.
